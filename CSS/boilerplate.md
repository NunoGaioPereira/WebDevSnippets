### CSS Boilerplate

CSS simple boilerplate ready to use.

```css
* {
 	margin: 0;
 	padding: 0;
 	box-sizing: border-box;
}

body {
	font-family: Arial, Helvetica, sans-serif;
}

a {
	color: #333;
	text-decoration: none;
}

ul {
	list-style: none;
}

table {
	border-collapse: collapse;
	border-spacing: 0;
}

.clearfix:after {
    content: ".";
    display: block;
    height: 0;
    clear: both;
    visibility: hidden;
}

```
